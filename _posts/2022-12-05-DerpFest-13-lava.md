DerpFest is a beautiful pixel rom. There are some features limited to Pixel devices like At a Glance Weather or Game Dashboard, which are now available to other AOSP devices. Some of the best additions by DerpFest are features like AOD icons on lockscreen, additions to Pulse like gravity options or round lines and optional beautiful handcrafted system icons

**First time installation:**

* Boot recovery
* Format Data, Wipe dalvik and cache
* Install ROM
* Install Magisk (optional)
* Reboot
